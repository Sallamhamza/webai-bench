# Security Policy

WebAI Bench stores no personal data by design (see `docs/06-security-privacy.md`). If you find
a vulnerability — including a way to bypass plausibility gates, exfiltrate more than the
documented schema, or de-anonymize a submission — please report it privately rather than opening
a public issue.

Report to: hamzaeng277@gmail.com

Please include: affected component, reproduction steps, and impact. We aim to acknowledge
reports within 5 business days.

The pre-launch security checklist lives in `docs/06-security-privacy.md` §7 and is tracked to
completion in `docs/security-testing.md` (added in E11).
